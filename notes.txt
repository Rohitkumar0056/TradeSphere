1. Setting up api-gateway service : all the api requests will first pass through this port, 
   It is used to connect all the services.

2. Setting up auth-service 

3. Create packages/error-handler/index.ts and error-middleware.ts
   In index.ts extend the error class and initialize all req errors
   In error-middleware.ts handle all the error created in the index.ts file

4. Setup and connect mongodb database
   Install prisma and create the schema in prisma/schema.prisma file
   Create packages/libs/prisma/index.ts file and create prismaClient

5. Create controller/auth.controller.ts file

6. Create utils/auth.helper.ts file and create all the req functions in it to import in the auth.controller.ts file

7. Setup and connect to redis
   Setup smtp variables
   
8. Create utils/email-template/user-activation-mail.ejs file and create an email template
   Create utils/sendMail/index.ts file and write nodemailer and transporter code in it which is used to send an email and use the user-activation-mail template
   Create sendOtp function in auth.helper and use sendMail function in it and send the randomly generated otp in the email

9. Create trackOtpRequests and checkOtpRestrictions function in auth.helper.ts to validate the otp and ensure security

10. Create a post route at /user-registration in routes/auth.router.ts file and pass userRegistration function from auth.controller in it

11. Setup swagger.js file in src folder and run this file
    It will automatically create a swagger-output.json file in the src folder

12. Setting up @ alias for importing files without writing long file address (many changes in the package files to implement it)
   
13. Create verifyUser, loginUser, forgotPassword, verifyForgotPasswordOtp, resetPassword functions in the auth.controller.ts and assign them auth.rotes.ts

14. Create the user-ui service, inside it in src/app update the layout and page.tsx files, and create the header.tsx file in src/shared/widgets/header/header.tsx and a index.ts to export the header file
    Create header-bottom file in the same folder and import it in the header
    Create a constants.tsx file inside configs folder to define title and links to be used in the header bottom.
    Create (routes)/login/page.tsx for login componenet, and similarly create signup page

15. Set up tanstack query in providers.tsx and import it in layout.tsx
